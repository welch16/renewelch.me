ChIPUtils

By Rene Welch in R genomics QC ChIP-seq

October 3, 2017

ChIPUtils is an R package that contains many functions used for exploratory analysis of ChIP-seq and ChIP-exo data.

It includes some commonly used quality control measures:

  • PCR bottleneck coefficient
  • Strand cross-correlation
